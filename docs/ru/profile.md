# Группировка профилей

Есть несколько способов сгруппировать свои профили:

### Объединение с помощью сервиса Яндекса
1. Зайдите на сайт social.yandex.ru
2. Расскажите о взаимных связях

{% note tip %}

Если у вас нет аккаунта на Яндексе, войдите на social.yandex.ru как пользователь социальной сети и разрешите поисковой системе использовать информацию о связях между профилями.

{% endnote %}

### Объединение с помощью перекрестных ссылок
1. Зайдите на сайт ВКонтакте, Мой Круг, Livejournal или ya.ru.
2. Добавьте ссылки на свои профили в других социальных сетях.

{% list tabs %}

- ВКонтакте

  Укажите профили в разделах **[Личный сайт](http://vk.com/edit?act=contacts)** и **[Интеграция с другими сервисами]( http://vk.com/edit?act=contacts)**.

- Мой Круг

  Укажите профили в разделе **[Я на других сайтах](http://moikrug.ru/master/profile/sites/)**.

- Livejournal
  
  Укажите профили в разделе **Способы связи (External services)**.

- ya.ru

  Укажите профили в разделе **Дневники**.

- Одноклассники
  
  Профили из социальной сети Одноклассники пока могут быть сгруппированы с другими профилями только с помощью [Яндекс.Паспорта](https://passport.yandex.ru/profile/social).

{% endlist %}

### Объединение с помощью подтвержденной ссылки
Добавьте подтвержденную ссылку в одной из социальных сетей. Подтвержденной считается ссылка, для публикации которой требуется ввести пароль другой сети.

{% note warning %}

Внесенные вами изменения будут учтены не моментально, а только при ближайшей индексации.

{% endnote %}
